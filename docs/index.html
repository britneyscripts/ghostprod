01 · the problem: why now?

In 2025, 58% of e-commerce searches are already mediated by agents. If your brand is invisible to GPT-5, it doesn't exist.

Semantic Invisibility

PDPs built only for visual rendering hide critical attributes from search and comparison agents.

Discovery Cost

Paying for traffic via Google Ads is getting expensive. The new organic frontier is being the #1 recommendation from ChatGPT.

THE SOLUTION

A scoring framework that analyzes the data pipeline of a PDP from the perspective of an intelligent crawler.

02 · how it works

Five specialist agents analyze your page in parallel.

AGENT 01

PageSpeed & CrUX

Measures real performance (CWV) via Chrome UX Report. Search agents penalize slow pages that drain tokens.

CrUX API PageSpeed BigQuery

AGENT 02

Schema.org Audit

Verifies the completeness and quality of JSON-LD. If data is not structured, it is noise for the agent.

JSON-LD Schema.org Validation

AGENT 03

Semantic Rendering

Renders the DOM in headless Chrome to hunt for dynamic content that WAFs block for simple crawlers.

Playwright Stealth DOM parser

AGENT 04

NLP Parseability

Uses spaCy to measure the density of real attributes vs. marketing noise. Can the agent extract color and size?

spaCy pt NLP regex

AGENT 05

Gap Analysis

Simulates real agent reasoning. Answers: with this data, would this product be recommended for "oily skin serum"?

Claude API FastAPI Python

04 · academic rigor: the methodology

∂

Thesis · ICMC/USP

MBA in Data Science. Hypothesis: the three variables have different weights in predicting semantic invisibility for Level 1 agents. ghostprod is the data collection instrument. Model: logistic regression + SHAP values.

◈

Substack Series

Agentic Commerce — articles with original data on how AI agents read Brazilian e-commerce. Two articles published. Each publication has verifiable data, a testable hypothesis, or a replicable experiment.

⌬

Field Data

CrUX API confirms: most individual PDPs do not reach the minimum Chrome traffic threshold to generate field data. This is a real methodological limitation — and a blind spot for all competitors using CrUX.
